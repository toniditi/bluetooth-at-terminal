"btTerminal" is FREE.

This application allows you to connect to DUN interface for phones that support AT commands.

Warning: To use this application You need to have knowledge of AT commands

What you need !

  1. Android device with Blluetooth
  2. Cell phone with Bluetooth

How to use it?

  • Start "btTerminal" on your Android device
  • It will get list of all "paired" devices
  • Select your "target phone" if present in list, or scan for it - (press "Scan for devices")
  • If "Send" button is enabled, the bluetooth interface connected.
  • Enjoy!
You can use this application for different purpose e.g. Read/Write phone book in target phone AT+CPBR/W, Read, Write or Send SMS AT+CMGR/W/S, Check battery charging level AT+CBC and etc.
